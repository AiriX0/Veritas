Veritas is a high-performance privacy and optimization engine designed for Windows environments. It provides a centralized interface for managing system hardening, deep-cleaning temporary assets, and securing sensitive files through steganography and military-grade shredding protocols.

##🔒 Features
System Shielding: Manage and toggle Windows telemetry, diagnostic data, location services, and DNS-over-HTTPS (DoH) settings.

Sanitizer: Perform deep-cleans of system scratchpad profiles, DNS caches, and browser storage databases to maintain digital hygiene.

File Vault: Utilize steganography to inject encrypted text directly into image file carriers, protected by a user-defined passkey.

Permanent Eraser: Securely shred files using customizable 1, 3, or 7-pass overwrite algorithms to prevent forensic recovery.

Perimeter Guard: Audit active network adapters, open socket connections, and firewall rule states.

##🚀 Installation & Usage
Veritas is built to be portable and autonomous.

Dependencies: The framework will automatically detect and install missing requirements (CustomTkinter and Pillow) upon the first launch.

##🛡️ Security Disclaimer
This tool is designed for advanced users and security professionals. Modifying registry keys and performing destructive file shredding can impact system stability. Always ensure you have valid backups of critical data before executing permanent erasure or system-level configuration changes.
