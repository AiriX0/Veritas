# 🛡️ Veritas

> A high-performance privacy and optimization engine designed for Windows environments.

Veritas is a comprehensive framework providing centralized control over system hardening, deep-cleaning temporary assets, secure file handling, and network auditing. Empower yourself with fine-grained control over your local environment's privacy footprint.

---

## ✨ Features

### 🔐 System Shielding
Manage and toggle Windows telemetry, diagnostic data, location services, and DNS-over-HTTPS (DoH) settings to enhance your privacy posture.

### 🧹 Sanitizer
Perform deep-cleans of system scratchpad profiles, DNS caches, and browser storage databases to maintain digital hygiene.

### 🗝️ File Vault
Utilize steganography to inject encrypted text directly into image file carriers, protected by a user-defined passkey. Keep sensitive data hidden in plain sight.

### 🗑️ Permanent Eraser
Securely shred files using customizable 1, 3, or 7-pass overwrite algorithms to prevent forensic recovery.

### 📡 Perimeter Guard
Audit active network adapters, open socket connections, and firewall rule states for comprehensive network visibility.

## 🚀 Installation & Usage

Veritas is built to be **portable and autonomous**.

### Prerequisites
- Windows OS
- Python 3.7+

### Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/AiriX0/Veritas.git
   cd Veritas
   ```

2. Run Veritas:
   ```bash
   python veritas.py
   ```

### Automatic Dependency Installation
The framework will automatically detect and install missing requirements on first launch:
- **CustomTkinter** - Modern UI framework
- **Pillow** - Image processing library

No manual setup required!
